# S&I Roofing Ltd Website

Website oficial pentru **S&I Roofing Ltd** - servicii profesionale de acoperișuri și construcții.

## Structură Proiect
- `index.html` - Pagina principală
- `styles.css` - Stiluri CSS moderne
- `script.js` - Logica și interactivitatea paginii

## Dezvoltare Locală
Deschide `index.html` într-un browser web sau rulează un server de dezvoltare local.
